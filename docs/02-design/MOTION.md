# Motion

## Hover

Cards:
- translateY(-2px)
- transition: 200ms ease-out

Buttons:
- active scale: 0.98

## Page Entry

- fade in
- translateY(16px to 0)
- duration: 300ms

## Loading

Use skeleton placeholders.

## Restrictions

- No bounce
- No excessive parallax
- No animated gradient backgrounds across full sections
