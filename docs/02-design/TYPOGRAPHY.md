# Typography

## Font

Recommended:
- Display: Inter, Geist, or Mona Sans
- Body: Inter

## Scale

```css
--font-display-xxl: 96px;
--font-display-xl: 72px;
--font-display-lg: 56px;
--font-display-md: 36px;
--font-title: 24px;
--font-card-title: 20px;
--font-body-lg: 18px;
--font-body: 15px;
--font-body-sm: 14px;
--font-caption: 13px;
--font-micro: 12px;
```

## Letter Spacing

```css
--tracking-display-xxl: -4.8px;
--tracking-display-xl: -3.6px;
--tracking-display-lg: -2.4px;
--tracking-display-md: -1.2px;
--tracking-body: -0.15px;
```

## Rules

- Hero titles should be large and compressed.
- Body copy should be short.
- Do not use too many font weights.
- Use size and spacing for hierarchy, not extreme boldness.
