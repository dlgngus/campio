# Components

## Button

### Primary

White pill button.

Use for:
- Get Started
- Continue
- Apply Now
- Save changes

Style:
- background: white
- color: black
- border-radius: pill
- min-height: 44px
- padding: 10px 16px

### Secondary

Dark charcoal pill.

Use for:
- Sign in
- Learn more
- Filter
- Cancel

### Text Link

Accent blue text only.

Use for:
- small navigation
- external apply links
- inline links

## Opportunity Card

Required content:
- category badge
- title
- organization
- deadline
- location/online
- saved count or popularity
- save button

States:
- default
- hover
- saved
- deadline soon

## Featured Opportunity Card

Use on Home for high-priority opportunities.

Required:
- bigger title
- stronger deadline
- recommendation reason
- Apply or View Details CTA

## Filter Chip

States:
- default
- selected
- disabled

Selected uses surface lift and tiny accent blue indicator, not full blue background.

## Input

- dark surface
- white text
- muted placeholder
- blue focus ring

## Empty State

Never write only "No data".

Use helpful copy:
- "No opportunities yet."
- "Try another category or clear filters."

## Skeleton

Use rectangular/card skeletons.
Avoid spinner-only loading.
