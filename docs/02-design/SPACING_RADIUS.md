# Spacing and Radius

## Spacing

Use this scale:

```css
--space-1: 4px;
--space-2: 8px;
--space-3: 12px;
--space-4: 16px;
--space-5: 24px;
--space-6: 32px;
--space-7: 48px;
--space-8: 64px;
--space-9: 96px;
```

## Radius

```css
--radius-sm: 8px;
--radius-md: 12px;
--radius-lg: 16px;
--radius-xl: 20px;
--radius-2xl: 28px;
--radius-pill: 999px;
```

## Usage

- Buttons: pill
- Inputs: 12px
- Opportunity cards: 20px
- Featured cards: 24px to 28px
- Gradient spotlight cards: 28px
