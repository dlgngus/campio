# Opportunity Detail

## Purpose

Give enough information to decide whether to apply.

## Header

- category
- title
- organization
- deadline
- save button
- apply now button

## Key Info Grid

- deadline
- location
- online/offline
- period
- target students
- benefits

## Main Sections

1. Description
2. Requirements
3. Benefits
4. Schedule
5. How to apply
6. Related opportunities
7. Related Q&A
8. Related mentors/reviews

## CTA

Primary: Apply Now
Secondary: Save

## Application Record

After clicking Apply Now, user can mark:
- Interested
- Preparing
- Applied
- Accepted
- Rejected
