# Explore Screen

## Purpose

Allow students to search and filter all opportunities.

## Header

```text
Explore opportunities.
```

## Search

Search by:
- title
- organization
- category
- tag

## Filters

- category
- deadline
- location
- online/offline
- grade
- major relevance
- saved
- application status

## Categories

- All
- Internship
- Contest
- External Activity
- Scholarship
- Exchange
- Startup
- Seminar
- Mentoring

## Desktop Layout

Left filter sidebar.
Right card grid.

## Mobile Layout

Top search.
Horizontal category chips.
Filter bottom sheet.
