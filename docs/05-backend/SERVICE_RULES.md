# Service Rules

## OpportunityService

Responsibilities:
- list opportunities
- filter opportunities
- get detail
- create/update/delete admin opportunity
- recommended list
- closing soon list
- popular list

## SavedOpportunityService

Responsibilities:
- save opportunity
- unsave opportunity
- list saved opportunities by user

## ApplicationRecordService

Responsibilities:
- create application record
- update status
- list records

## CommunityService

Responsibilities:
- list posts
- create post
- get post detail
- add comment

## MentorService

Responsibilities:
- list mentors
- mentor detail
- create mentor profile
