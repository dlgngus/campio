# User Flow

## First-time User Flow

```text
Landing
→ Signup
→ School Verification
→ Major Selection
→ Interest Selection
→ Home
```

## Returning User Flow

```text
Login
→ Home
→ Recommended Opportunity
→ Opportunity Detail
→ Save or Apply
```

## Discovery Flow

```text
Home
→ Category Section
→ Explore
→ Filter
→ Opportunity Detail
→ Save
→ Apply Now
```

## Deadline Flow

```text
Home
→ Closing Soon
→ Detail
→ Save
→ Add Application Record
```

## Community Support Flow

```text
Opportunity Detail
→ Related Q&A
→ Community Post
→ Comment
```

## Mentor Support Flow

```text
Opportunity Detail
→ Related Mentor
→ Mentor Profile
→ Ask Question
```

## Admin Flow

```text
Admin Login
→ Admin Opportunities
→ Create Opportunity
→ Publish
→ Home/Explore Feed
```
